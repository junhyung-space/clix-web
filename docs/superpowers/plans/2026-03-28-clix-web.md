# CLIX Web Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** clix macOS 앱을 소개하고 GitHub Releases 다운로드를 제공하는 단일 스크롤 랜딩 페이지를 Astro + Tailwind로 구축하고 GitHub Pages에 자동 배포한다.

**Architecture:** 단일 `index.astro` 페이지에 8개 섹션 컴포넌트를 세로로 배치하는 단일 스크롤 구조. i18n은 `src/i18n/ko.ts` / `src/i18n/en.ts` 번역 파일 + 클라이언트 사이드 JS로 처리 (페이지 리로드 없이 전환). 언어 선택은 `localStorage`에 저장하고, 페이지 로드 시 브라우저 언어를 감지해 초기값으로 사용.

**Tech Stack:** Astro 4, Tailwind CSS 3, TypeScript, GitHub Actions, GitHub Pages

- [ ] **Step 1: 워크플로우 파일 작성**

```bash
mkdir -p .github/workflows
```

`.github/workflows/deploy.yml`:

```yaml
name: Deploy to GitHub Pages

on:
  push:
    branches: [main]
  workflow_dispatch:

permissions:
  contents: read
  pages: write
  id-token: write

concurrency:
  group: pages
  cancel-in-progress: true

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: npm
      - run: npm ci
      - run: npm run build
      - uses: actions/upload-pages-artifact@v3
        with:
          path: dist/

  deploy:
    needs: build
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url: ${{ steps.deploy.outputs.page_url }}
    steps:
      - uses: actions/deploy-pages@v4
        id: deploy
```

    <script>
      import { ko } from '../i18n/ko';
      import { en } from '../i18n/en';

      type Lang = 'ko' | 'en';

      function detectLang(): Lang {
        const saved = localStorage.getItem('clix-lang') as Lang | null;
        if (saved === 'ko' || saved === 'en') return saved;
        return navigator.language.startsWith('ko') ? 'ko' : 'en';
      }

      function applyLang(lang: Lang) {
        const t = lang === 'ko' ? ko : en;
        document.querySelectorAll<HTMLElement>('[data-i18n]').forEach((el) => {
          const key = el.dataset.i18n!;
          const value = key.split('.').reduce((obj: any, k) => obj?.[k], t);
          if (typeof value === 'string') el.textContent = value;
        });
        document.documentElement.lang = lang;
        localStorage.setItem('clix-lang', lang);

        // Update toggle button
        const toggle = document.getElementById('lang-toggle');
        if (toggle) toggle.textContent = lang === 'ko' ? 'KO / EN' : 'EN / KO';
      }

      // Init on load
      const currentLang = detectLang();
      applyLang(currentLang);

      // Toggle on click
      document.getElementById('lang-toggle')?.addEventListener('click', () => {
        const next: Lang =
          (localStorage.getItem('clix-lang') as Lang) === 'ko' ? 'en' : 'ko';
        applyLang(next);
      });
    </script>
